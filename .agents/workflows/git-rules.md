# Git Rules

- **Do NOT commit or push unless the user explicitly asks you to.**
- You may stage files (`git add`) for review, but do not run `git commit` or `git push` on your own.
- When the user says "commit" or "commit and push", then proceed.
- **Each "commit" or "push" instruction applies ONLY to that specific request. It does NOT carry over to subsequent requests. Every new request starts with NO permission to commit or push. You must wait for explicit instruction each time.**
